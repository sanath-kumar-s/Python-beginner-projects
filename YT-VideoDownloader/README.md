## 📖 About

This is a command-line YouTube downloader built in Python that allows users to download videos, audio, thumbnails, or entire playlists with control over quality. It focuses on being simple to use while still offering flexibility like manual quality selection and automatic merging of video and audio streams.

---

## ✨ Features

- Download full videos in selected quality
- Auto-download best quality (video + audio merged)
- Extract and download audio as MP3
- Download video thumbnails only
- Supports playlist downloads
- Displays available video quality options (resolution-based)
- Shows real-time download progress (percentage + speed)
- Automatically creates and manages a downloads folder
- Handles errors and skips unavailable items in playlists

## 🚀 Getting Started

### Prerequisites

Before you begin, ensure you have the following installed:

- Python 3.10 or higher
- pip (Python package manager)

### Installation

1. **Clone the repository**

```bash
git clone https://github.com/sanath-kumar-s/Python-beginner-projects.git
cd YT-VideoDownloader
```

2. **Install dependencies**

```bash
pip install -r requirements.txt
```

3. **Run a project**

```bash
cd YT-VideoDownloader
python main.py
```
